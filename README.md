# Ciptacode Video Downloader (CVD)

A clean, ad-free, and open-source YouTube video downloader built with Python. This desktop application features a modern graphical user interface and utilizes the powerful `yt-dlp` engine to download videos and extract audio seamlessly.

## Features

* **Multiple Download Options:** Choose between Video + Audio, Video Only, or Audio Only (MP3).
* **Resolution Selection:** Support for various resolutions including Best Quality, 1080p, 720p, 480p, and 360p.
* **Modern Interface:** Built with CustomTkinter, featuring a sleek, automatic dark mode design.
* **Real-Time Tracking:** Includes a live progress bar and status indicator to track your download progress without freezing the application.
* **Custom Save Location:** Easily browse and select the destination folder for your downloaded files.
* **Anti-Playlist Protection:** Automatically prevents accidental massive bulk downloads from YouTube Mixes or playlists.

## Prerequisites

To run this application from the source code, you will need the following installed on your system:

1. **Python 3.x**
2. **FFmpeg:** Strictly required for extracting and converting downloaded media into MP3 format. Ensure FFmpeg is downloaded and its `bin` folder is added to your system's Environment Variables (Path).
